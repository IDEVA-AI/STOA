# Desenvolvimento - Julio Carvalho

## Diretriz Principal
- **Foco exclusivo na Interface (Frontend/UI/UX).**
- **Não se preocupar com a lógica de Backend/Banco de Dados.**
- Priorizar estética editorial, tipografia e interações premium.

## Notas de Design
- Tipografia: Serifada para títulos (Playfair Display/Georgia), Sans para corpo (Inter).
- Cores: Paleta sóbria (Preto, Branco, Ouro, Cinza Quente).
- Mood: Profissional, Elite, Arquitetura de Sistemas.
